# block-mute — acceptance criteria

> See [overview.md](overview.md) for goals.

## Criteria

- [ ] ローカルアカウントをブロックでき、一覧・解除できる
- [ ] リモートアカウントをブロックでき、相手 inbox に Block アクティビティが配送される(E2E で検証)
- [ ] ブロック時に双方向のフォロー関係が解除される
- [ ] inbox で受信した Block がフォロー関係の解除に反映される
- [ ] ミュートはローカルのみで機能し、一覧・解除できる
- [ ] Mock peer / Iceshrimp E2E にブロックのシナリオが追加される
